### Installation

Install Voltx globally using your preferred package manager

```package-install
npm i -g voltx
```

### Initial Setup

After installation, you'll need to initialize VoltX with your preferred configuration:

```bash
voltx init
```

<Callout title="API Key Required">
	You must configure at least one AI provider with a valid API key during the
	`init` process to use VoltX's core features like chat and agent mode.
</Callout>

This command will guide you through the setup process:

<Steps>

<Step> Create a User Alias </Step>
<p>Enter a unique username for your VoltX profile.</p>

<Step> Select AI Provider </Step>
<p>Choose the first AI service you want to connect with.</p>

<Step> Enter API Key </Step>
<p>Paste the API key for the selected provider.</p>

<Step> Select Default Model </Step>
<p>Pick the default AI model you want to use with this provider.</p>

  <Step> Configure More Providers (Optional) </Step>
  <p>Choose whether to add another AI provider now or finish setup.</p>
</Steps>

## Configuration Structure

VoltX creates a configuration directory structure in your home directory:

## VoltX Help

To verify your installation and see available commands:

```bash
voltx help
```
